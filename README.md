# Proyecto JavaScript

Aplicación básica en JavaScript con Git, ESLint, pre-commit, Docker y Docker Compose.

## Ejecutar localmente

npm start

## Ejecutar ESLint

npm run lint

## Ejecutar pre-commit

pre-commit install
pre-commit run --all-files

## Ejecutar con Docker Compose

docker compose up --build
